## API References

| Prop      | Type                           | Description        | Default |
| :-------- | :----------------------------- | :----------------- | :------ |
| size      | `"xs"\|"sm"\|"md"\|"lg"\|"xl"` | Size of the Radio  | sm      |
| color     | `string`                       | Color of the Radio | blue    |
| className | `string`                       | className          | \_      |
