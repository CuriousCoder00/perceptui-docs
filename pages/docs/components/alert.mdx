## API References

| Prop      | Type                                          | Description          | Default |
| :-------- | :-------------------------------------------- | :------------------- | :------ |
| variant   | `"solid" \| "outline" \| "surface`            | Form of the Alert    | solid   |
| type      | `"success" \| "warning" \| "error" \| "info"` | Type of the Alert    | success |
| message   | `string`                                      | Message of the Alert | \_      |
| icon      | `ReactNode`                                   | Icon of the Alert    | type    |
| className | `string`                                      | className            | \_      |

## Examples

### Variant

Use the `variant` prop to control the visual style of the button.

<div className="flex flex-col my-6 px-2 w-full">
  <div className="flex flex-col gap-2 p-4 bg-blue-950/15 mb-2 rounded-lg w-full">
    <Alert variant={"solid"} message="This is alert with solid variant." />
    <Alert variant={"outline"} message="This is alert with outline variant." />
    <Alert variant={"surface"} message="This is alert with outline variant." />
  </div>
  <div className="block text-wrap w-full py-1">
