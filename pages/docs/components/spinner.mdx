## API References

| Prop      | Type     | Description | Default |
| :-------- | :------- | :---------- | :------ |
| color     | `string`                       | Color of the Spinner | blue    |
| size      | `"xs"\|"sm"\|"md"\|"lg"\|"xl"` | Size of the Spinner  | md      |
| className | `string`                       | className          | \_      |
