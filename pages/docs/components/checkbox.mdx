## API References

| Prop      | Type                           | Description           | Default |
| :-------- | :----------------------------- | :-------------------- | :------ |
| size      | `"xs"\|"sm"\|"md"\|"lg"\|"xl"` | Size of the Checkbox  | sm      |
| color     | `string`                       | Color of the Checkbox | blue    |
| className | `string`                       | className             | \_      |
