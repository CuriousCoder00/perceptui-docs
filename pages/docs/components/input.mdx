## API References

| Prop      | Type                                     | Description         | Default |
| :-------- | :--------------------------------------- | :------------------ | :------ |
| variant   | `"classic" \| "standard" \| "ghost"`     | Form of the Input   | classic |
| radius    | `"none"\|"sm"\|"md"\|"lg"\|"xl"\|"full"` | Radius of the Input | lg      |
| className | `string`                                 | className           | \_      |

## Examples
